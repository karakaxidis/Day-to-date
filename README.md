# Μετατροπέας Αριθμού Ημέρας → Ημερομηνία (PWA)

Μικρή εφαρμογή που μετατρέπει τον **αριθμό της ημέρας** (1–365/366) σε **ημερομηνία** για συγκεκριμένο έτος.  
Υλοποιημένη ως **Progressive Web App (PWA)** ώστε να μπορεί να εγκατασταθεί σε κινητό και να λειτουργεί offline.

---

## 🚀 Χρήση
1. Άνοιξε την εφαρμογή από τον browser σου.
2. Δώσε **έτος** και **αριθμό ημέρας**.
3. Πάτησε **Μετατροπή** → εμφανίζεται η αντίστοιχη ημερομηνία.
4. Πάτησε **Υπολογισμός σημερινής ημέρας** για να βρεις σε ποια ημέρα του έτους βρισκόμαστε.

---

## 📱 Εγκατάσταση σε κινητό
Αφού ανέβει το site:

- **Android (Chrome)**  
  1. Menu ⋮ → **Add to Home screen** → Add.  
  2. Θα εμφανιστεί σαν αυτόνομη εφαρμογή.  

- **iPhone (Safari)**  
  1. Share (τετράγωνο με βελάκι) → **Add to Home Screen** → Add.  
  2. Θα εμφανιστεί σαν app με δικό της εικονίδιο.  

Μετά την πρώτη επίσκεψη, λειτουργεί **offline**.

---

## 🛠 Ανάπτυξη με GitHub Pages
1. Δημιούργησε νέο repo (π.χ. `day-to-date-pwa`).
2. Ανέβασε τα αρχεία:  
   - `index.html`  
   - `manifest.webmanifest`  
   - `sw.js`  
   - φάκελο `icons/`
3. Πήγαινε στο **Settings → Pages**.
4. Στην ενότητα **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: `main` (ή `master`)  
   - Folder: `/ (root)`
